# Contributing to Halo

Halo is an experimental personal project. Contributions and ideas are welcome.

Full contributor guide: **[docs/development/contributing.md](docs/development/contributing.md)**

Quick version:

```bash
git clone https://github.com/CK-Technology/halo.git
cd halo
cargo build
cargo test
cargo fmt && cargo clippy --all-targets
```

See the [roadmap](docs/development/roadmap.md) for what exists today versus what is still
design intent, and the [documentation index](docs/README.md) for the architecture.
